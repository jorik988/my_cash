MyCash - это приложение для управления финансами, которое позволяет пользователям отслеживать свои доходы и расходы.

Основные функции:
1. Добавление записей: Возможность добавлять записи о своих доходах и расходах, указывая дату, категорию (Доход или Расход), сумму и описание.
2. Редактирование записей: Возможность редактировать существующие записи, изменяя дату, категорию, сумму или описание.
3. Поиск по записям: Возможность искать записи по категории, дате или сумме. Поиск поддерживает форматы даты ГГГГ-ММ-ДД, ГГГГ-ММ и ГГГГ.
4. Вывод баланса: Приложение может выводить общий баланс пользователя, а также общие доходы и расходы.
5. Поддержка нескольких пользователей: Данные каждого пользователя сохраняются в отдельном файле %username%.json и загружаются при запуске.
